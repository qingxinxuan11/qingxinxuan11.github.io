JS之基础

JS 基础知识之组成

1、ECMAscript－提供核心语言的功能性和基础
2、Dom－提供访问和操作网页的内容和方法的接口
3、Bom－提供与浏览器交互的方法和接口

JS 基础之数据类型

1、undefined：未定义的
2、number：数字类型
3、string：字符串类型
4、boolean：布尔类型
5、function：函数类型
6、object：对象类型包括数组、null、字符串

JS 基础之标示符

标示符指的是变量、函数、属性的名字或者函数的参数

命名首字符必须是字母、下划线、$、不能有空格且不能为关键字和保留字

JS 基础之运算符

1、赋值运算符：=
2、算术运算符：= - * /
3、关系操作符：> < == <= >=
4、逻辑运算符：&&(与) ||(或) !(非)
5、递增递减：++ – ++i先运算再赋值 i++先赋值再运算

JS 基础之操作优先级

罗技非>算数操作符>逻辑与或>条件运算>赋值

JS Boolean真假判断

true:非零数字，非空字符串

false:数字0，空字符串“”，空对象null，undefined，NaN
